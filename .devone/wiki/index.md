# DevOne Wiki

本项目知识库总入口。

## 目录结构

| 目录 | 用途 |
|------|------|
| [projects](./projects/) | 项目相关知识 |
| [technologies](./technologies/) | 技术栈、工具、框架 |
| [patterns](./patterns/) | 模式与最佳实践 |
| [decisions](./decisions/) | 架构决策记录 (ADR) |
| [references](./references/) | 参考资料、链接 |

## 快速开始

1. 阅读 [projects/README.md](./projects/README.md) 了解项目概况
2. 查阅 [technologies/README.md](./technologies/README.md) 了解技术栈
3. 参考 [patterns/README.md](./patterns/README.md) 了解编码规范

## 知识贡献原则

1. **最小化**：使用最少的必要词汇
2. **准确性**：基于证据，避免猜测
3. **可追溯性**：包含来源和引用
4. **一致性**：保持命名和分类稳定
5. **时效性**：定期检查和更新
